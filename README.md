LaunchNow

Forked from ggkevinnnn/LaunchNow – Alternative Launchpad application for macOS Tahoe.

Features<br>
	•	Tahoe style and classic style supported<br>
	•	Create folders<br>
	•	Rearrange icons<br>
	•	Search apps<br>
	•	Keyboard navigation<br>
	•	Open source<br>

New Features in this fork:<br>
	•	Default LaunchNow state starts with no apps<br>
	•	Add apps via Settings → Import<br>
	•	Multi-select import supported<br>
	•	Reset Apps button to clear all added apps<br>

Usage<br>
	1.	Download the app<br>
	2.	(Optional) Add LaunchNow to login items<br>
	3.	Drag apps to customize application sort or create folders<br>
	4.	Press Down Arrow or Return on keyboard to enable full keyboard navigation<br>
	5.	Scroll to turn the page<br>
	6.	Use Settings → Import to add apps<br>
	7.	Use Reset Apps button to clear all apps<br>
