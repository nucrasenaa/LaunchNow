LaunchNow

Forked from [ggkevinnnn/LaunchNow](https://github.com/ggkevinnnn/LaunchNow) – Alternative Launchpad application for macOS Tahoe.

Features in this fork:<br>
Clear sectioned settings layout<br>
   • Appearance & Interaction<br>
   • Grid Layout<br>
   • App Management<br>
   • App Settings Backup<br>
   • Maintenance<br>

Consistent left-aligned section headers<br>
   • Unified horizontal padding at the container level<br>
   • Clean visual separation via dividers<br>

Appearance & Interaction controls<br>
   • Toggle Classic Launchpad (Fullscreen) mode<br>
   • Adjustable scrolling sensitivity (Low ↔ High)<br>
   • Background run guidance message<br>

Grid layout customization<br>
   • Adjustable Columns (3–12)<br>
   • Adjustable Rows (2–8)<br>
   • Auto-updating Items per page display<br>
   • Immediate grid refresh on change<br>

App management tools<br>
   • Add App (renamed from Import App) with selection sheet and search<br>
   • Remove App with selection sheet, search, and option to include folder apps<br>
   • Reset App action with safety confirmation (clears apps, folders, layout only in Launchpad)<br>

App settings backup<br>
   • Import App Setting (renamed from Import Data)<br>
   • Export App Setting (renamed from Export Data)<br>
   • Validation of imported data folder (SwiftData store check)<br>

Maintenance actions<br>
   • Reset Layout with confirmation (rescans apps, preserves disk apps; won’t auto-add)<br>
   • Quit application<br>

UX and behavior details<br>
   • Preserves existing import/remove sheet flows<br>
   • Keeps alerts and confirmations for destructive actions<br>
   • Triggers grid refreshes and order persistence where needed<br>
   • Pre-scan available apps on appear for faster Add App experience<br>

Usage<br>
	1.	Download the app<br>
	2.	Drag the app to Applications<br>
	3.	Go to System Settings → Privacy & Security → Allow app from unidentified developer<br>
	4.	(Optional) Add LaunchNow to login items<br>
	5.	Use Settings → Import to add apps<br>
	6.	Drag apps to customize sort order or create folders<br>
	7.	Press Down Arrow or Return on keyboard to enable full keyboard navigation<br>
	8.	Scroll to turn the page<br>
	9.	Use Reset Apps button to clear all apps<br>
